## 官方账号无法访问 /admin 修复
- 根因：web/app 的 `AdminPage` 守卫在 `useEffect` 中直接判断 `!user || !['admin','official'].includes(user.role)` 并 `navigate('/')`；但 `AuthContext` 初始 `user` 为 null（登录态异步校验），首帧即被误踢回首页——官方/管理员在 user 尚未就绪时进入都会被弹走。
- 修复：两个 `AdminPage` 改用 `const { user, loading: authLoading } = useAuth();`，守卫前加 `if (authLoading) return;` 等登录态就绪再判断。后端 `adminRequired` 本就放行 official（rank≥2），无需改。
